This repository is a place to throw wxGlade `.wxg` (XML project files) and working Perl script drivers to generate the GUI that are directly from wxGlade.
These are to be used as examples and starting points for wxPerl projects using wxGlade.
To set up the wxPerl environment on Windows, please see this page:

https://wiki.perl-guilds.net/index.php?title=Developing_and_Distributing_wxPerl_Applications_on_Windows
